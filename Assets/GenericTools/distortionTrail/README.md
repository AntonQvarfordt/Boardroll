Options
-------------------------------------

**_Distortion**  
This is how much distortion that Trail Makes, between 0 and 10.

**_NormalMap**  
This is a texture that controls how the render works.

**_UseDepth**  
Some platforms cannot calculate the depth, for those platforms turn this toggle off.

-------------------------------------
